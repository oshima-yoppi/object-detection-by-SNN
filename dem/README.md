
# イベント駆動コンピューティングによる月面の障害物検知

月面クレータをイベントカメラ使って検知したい。  
1. Matlabで月面DEMを生成
2. Blenderで月面DEMを再現し、アニメーションを行い、動画を作成する
3. V2Eを用いて動画からイベントデータに変換する。
4. snntorchで学習を行う
